# Installation:
## Linux / Windows
```console
git clone https://github.com/

cd Discord-Nuke-Bot

python -m pip install -r requirements.txt

(run this file)
start.bat


```

***


# Running
It's that simple to run! Just Enter the command `python nuke-bot.py` or `python3 nuke-bot.py` and the nuke bot Will be Launched
